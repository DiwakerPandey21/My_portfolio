# DP_OS v4 — Interactive OS Portfolio 🚀

Welcome to **DP_OS v4**, my personal and professional portfolio designed as a fully functional, interactive, web-based Operating System. Built entirely from scratch using Vanilla web technologies, this portfolio offers visitors a unique, immersive experience to explore my software engineering journey.

## ✨ Features

- **Cinematic Hacker Boot Sequence**: Experience an ultra-fast, data-decrypt-style terminal loading sequence that glitches into reality.
- **Authentic OS UI**: Features an advanced draggable window system, a functional Top Menu Bar, and an interactive macOS-style App Dock with 'Genie' minimization animations.
- **Dynamic Canvas Visuals**: Powered by the HTML5 `<canvas>` API, the dynamic Black & White halftone grid wallpaper mathematically maps a pixel-perfect portrait using thousands of reactive dots.
- **Interactive Web Apps**:
  - `Terminal`: Parses real commands like `whoami`, `skills`, `clear`, or find the hidden *hire* easter egg.
  - `System Monitor`: Real-time hardware integration displaying your actual device battery level using the native Browser API.
  - `Music Player` & `Snake Game`: Functional, playable applications directly on the desktop.
- **Mobile Responsive**: The complex OS layout gracefully adapts into a sleek, swipeable, and perfectly spaced mobile view on devices under 768px width.
- **Native Web Audio FX**: Browser-native synthesized UI sound effects powered entirely by mathematical `AudioContext` oscillators (no slow audio files!).

## 🛠️ Technology Stack

No heavy frameworks. Just pure, performant code:
- **HTML5** & **CSS3**: Advanced CSS Flexbox/Grid, responsive `@media` queries, fluid `@keyframes` animations, and glassmorphism.
- **Vanilla JavaScript (ES6+)**: Complete DOM Manipulation, state management, complex window-dragging constraints, event listeners, and API hooking.
- **HTML5 Canvas API**: High-performance pixel manipulation and visual rendering for the desktop backdrop.

## 🚀 Running Locally

Because DP_OS v4 is built with straightforward web technologies, there is no complex build step or transpilation required!

1. Clone the repository:
   ```bash
   git clone https://github.com/DiwakerPandey21/My_portfolio.git
   ```
2. Navigate into the project directory:
   ```bash
   cd My_portfolio
   ```
3. Use any local development static server to run it. For example, using Node.js:
   ```bash
   npx serve .
   ```
   Or using Python:
   ```bash
   python3 -m http.server
   ```
4. Open your browser and go to `http://localhost:3000` (or the port specified by your server).
5. **Login Hint:** Use the password `123a` to unlock the computer!
